▼本コンポーネントについて
  いくつかの拡張変数を使えるようにします

  --------------------------------------------------------------------------------------------------------------------------------------------------------
  変数名               解説
  --------------------------------------------------------------------------------------------------------------------------------------------------------
  %file_created%       ファイルの作成日時を表示します
  %last_accessed%      ファイルの最終アクセス日時を表示します
  %url_prefix%         パスのurl_prefixを表示します 例: file, unpack, http
  %is_file%            パスのurl_prefixがfileであるかどうかを返します $strcmp(%url_prefix%,file)と同じです
  %is_archive%         foobar2000のAPI archive_impl::g_is_unpack_pathの結果を返します 書庫内のファイルであるかどうかが判定できます
  %archive_root%       例: C:\Users\foobar\Desktop\test_archive.zip内のtest.mp3ファイルに使用した場合、C:\Users\foobar\Desktop\test_archive.zipと返します
  %archive_branch%     上記の例で test_archive/test.mp3 と返します
  %archive_type%       archiveのタイプを返します 例: zip, 7z
  --------------------------------------------------------------------------------------------------------------------------------------------------------

  * %file_created%と%last_accessed%について
    書式は yyyy-mm-dd hh:mm:ss です
    Title Formattingの $date や $time などを使ってお好みの書式に変更してください
    書庫内のファイルの場合は親フォルダ?の情報を参照します
    例えば C:\archive.zip 内の hoge.mp3 というファイルの場合、C:\archive.zipの作成日時と最終アクセス日時が取得されます
    これは組み込みの変数である %last_modified% と(たぶん)同じ仕様です
    Masstaggerの Format value from other fields と合わせて使うと タグに埋め込めます

  * %archive_root%について
    ファイルを移動した場合に値が更新されないかもしれないので注意してください
